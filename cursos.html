DESENVOLVIMENTO DE SISTEMAS

Formação completa para quem deseja atuar como desenvolvedor em múltiplas plataformas:

  • Tecnologias abordadas: HTML, CSS, JavaScript (incluindo frameworks como React), Python, Java, C#, e uso de APIs.
  • Desenvolvimento multiplataforma: Web, mobile (Android/iOS com Flutter ou React Native), e aplicações desktop.
  • Banco de dados: MySQL, PostgreSQL, MongoDB, e integração com sistemas.
  • Práticas de mercado: Git e GitHub, testes automatizados, DevOps e integração contínua (CI/CD).
  • Projetos reais: Desenvolvimento de sistemas em equipe com metodologias ágeis (Scrum, Kanban).
  • Preparação para certificações: Oracle Java, Microsoft .NET, e certificações web.

REDES DE COMPUTADORES

Capacitação técnica para atuar na instalação, manutenção e gerenciamento de redes:

  • Infraestrutura de redes: Cabeamento estruturado, topologias, protocolos de comunicação.
  • Dispositivos de rede: Configuração de switches, roteadores, firewalls e access points.
  • Sistemas operacionais de rede: Windows Server, Linux (Ubuntu, CentOS), AD, DNS, DHCP.
  • Segurança e monitoramento: Wireshark, Nmap, pfSense, SNMP e Nagios.
  • Simulações e práticas: Cisco Packet Tracer e GNS3.
  • Certificações visadas: CompTIA Network+, Cisco CCNA.

CIÊNCIA DE DADOS

Formação voltada para profissionais que desejam transformar dados em decisões estratégicas:

  • Programação aplicada: Python (Pandas, NumPy, Scikit-learn), R, SQL.
  • Estatística e matemática aplicada: Probabilidades, distribuições, regressões e modelagem preditiva.
  • Machine Learning e IA: Algoritmos supervisionados e não supervisionados, redes neurais.
  • Visualização de dados: Power BI, Tableau, Matplotlib, Seaborn.
  • Trabalho com grandes volumes de dados: NoSQL, Big Data, análise em nuvem (Google Colab, AWS).
  • Projetos práticos: Dashboards, análise de dados reais, desafios de dados.

SEGURANÇA DA INFORMAÇÃO

Formação sólida para atuar na proteção de sistemas e dados corporativos:

  • Fundamentos de segurança: Confidencialidade, integridade e disponibilidade (CIA).
  • Pentest e testes de vulnerabilidades: Metasploit, Burp Suite, OWASP ZAP.
  • Segurança de redes e sistemas: Firewalls, VPNs, IDS/IPS, criptografia, backups, autenticação forte.
  • Normas e políticas: ISO/IEC 27001, LGPD, GDPR, gestão de riscos e compliance.
  • Resposta a incidentes: Análise de logs, forense digital, mitigação de ataques.
  • Certificações visadas: CompTIA Security+, CEH (Certified Ethical Hacker).
